이항 연산자

연산자 설명 예시
+ 덧셈 연산자 2 + 3
- 뺄셈 연산자 7 - 2
* 곱셈 연산자 5 * 1
/ 나눗셈 연산자 35 / 7
% 나머지 연산자 41 % 6
** 제곱 연산자 3 ** 2

단항 연산자

연산자 설명 예시
+ 단항 덧셈 연산자 +2
+'3'
- 단항 뺄셈 연산자 -(-3)

할당 연산자

연산자 설명 예시
= 할당 연산자 let x = 5

복합 할당 연산자

연산자 설명 예시
+= 할당 연산자 a += 2
-= 할당 연산자 b -= 1

연산이 시행되고 난 후의 결과가 변수에 할당
%, **과 같은 다른 산술 연산자나 비트 연산자로도 사용가능

증감 연산자

연산자 설명 예시
++ 증가 연산자 a++
++a
– 감소 연산자 b--
--b

변수에만 사용 가능하며, 값에 사용하면 오류가 발생
전위형의 경우 변화시킨 값을 사용할 때, 후위형은 기존의 값을 사용한 뒤 값을 변화시킬 때 사용
